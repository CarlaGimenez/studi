La formation « Graduate Développeur Mobile IOS » vous permet d'acquérir les compétences pour créer des applications mobiles pour IOS, le système d'exploitation d'Apple et des iPhones.

Concevez des applications performantes pour l'iPhone et l'iPad !

Le programme de la formation s'articule autour de 3 axes :

1. Le Développement Front-End
  • Réaliser la maquette d'une application web responsive
  • Créer un site web avec HTML, CSS et le Framework Bootstrap
  • Installer Git
  • S'initier à la programmation avec Javascript
  • Développer une application web avec React.js
  • Créer un site web avec le CMS Wordpress
2. Le Développement Back-End
  • Créer et administrer une base de données
  • Développer des sites web dynamiques avec PHP
  • Créer une application web avec le framework Symfony
  • Développer une application mobile avec React Native
  • Créer un plugin Wordpress
3. Spécialisation Développement Mobile IOS
  • Programmer en Swift
  • Utiliser SwiftUI
  • Construire une application robuste et évolutive
  • Comprendre UIKit
  • Accéder aux fonctionnalités système

Cette formation vous permet d'acquérir toutes les compétences nécessaires pour donner vie à des sites web et applications mobiles pour IOS.

À l'issue de la formation, vous serez en mesure de :

  • Analyser un cahier des charges et choisir la solution technique adaptée
  • Concevoir un site ou une application côté front-end et back-end
  • Créer des applications mobiles IOS
