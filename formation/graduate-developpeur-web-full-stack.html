Lancez-vous dans le monde du développement web !

Donnez vie à des sites internet optimisés et épanouissez-vous dans un secteur en plein essor vous offrant des perspectives de carrières en France et dans le monde entier !

La formation « Graduate Développeur Web Full Stack » vous permet d'acquérir toutes les compétences nécessaires pour être capable de gérer la création d'un site web à la fois côté utilisateur (front-office) et côté administrateur (back-office).

Le programme s'articule autour de 2 grands axes :

1. Le Développement Front-End
  • Réaliser la maquette d'une application web responsive
  • Créer un site web avec HTML, CSS et le Framework Bootstrap
  • Utiliser Git et Github pour gérer son code source
  • S'initier à la programmation avec Javascript
  • Développer une application web avec React.js
  • Créer un site web avec le CMS Wordpress
2. Le Développement Back-End
  • Comprendre la stratégie d'entreprise
  • Définir ses Buyers Personas
  • Définir une stratégie d'acquisition ou de fidélisation et les KPI associés
  • Définir sa stratégie de content

Les développeurs sont les profils les plus demandés sur le marché de l'emploi, tous métiers confondus. Entrer dans ce domaine, c'est pour vous la garantie de trouver un emploi valorisant avec une rémunération attractive en France ou à l'étranger, en entreprise ou en home office.

Cette formation vous permet d'acquérir des compétences dans la création de sites internet optimisés et de vous épanouir dans un secteur en plein essor.

Vous apprendrez les langages nécessaires au développement d'interfaces web (HTML5, CSS3, Ajax, React.js, etc.), vous serez également capable de créer des pages web dynamiques (PHP, Symfony) et vous maîtriserez parfaitement les bases de données SQL. C'est une formation complète tournée uniquement vers les enseignements professionnels et dispensée par des formateurs experts et des professionnels reconnus.

À l'issue de la formation, vous serez en mesure de :

  • Élaborer des chartes graphiques
  • Créer des sites internet ou des applications mobiles
  • Créer et structurer des bases de données
  • Construire un back-office
  • Développer différentes fonctionnalités

Le développeur web donne vie aux sites internet et aux applications web mobiles à partir de maquettes visuelles. Il gère bien entendu l'intégration des contenus en HTML5 & CSS3 mais doit aussi imaginer le fonctionnel et structurer les interactions du site. Il est le garant du bon fonctionnement des interfaces sur les différents navigateurs et les différentes devices (dont le mobile). Il gère également la relation client-serveur ainsi que la ou les bases de données du site.

La formation de développeur web full stack vous permettra d'exercer rapidement un grand nombre de métiers dans l'environnement du code et du développement :

  • Intégrateur web
  • Développeur web full stack junior
  • Webmaster
  • Développeur JavaScript
